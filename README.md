# 🧐 요구사항 및 만족 

> Django 프로젝트 생성  

> web application 서버는 NGINX 서버를 사용합니다. (80번 포트)  
* docker-compose 파일로 nginx와 web 서버를 구축하였습니다.

> Elastic IP를 ec2에 할당합니다

> IP/api/hello 로 접속하였을 때 웹브라우져 상에서 Hello 표시합니다.
